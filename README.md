# Code and Supplements for "A Practical Guide to Data Analysis Using R"

Here is made available code and other supplementary content that relates to the new Cambridge University Press text, due to appear in May or June 2024:\
"A Practical Guide to Data Analysis Using R -- An Example-Based Approach",\
by John H Maindonald, W John Braun, and Jeffrey L Andrews.[^readme-1]
